# Machine-Learning

# 📉 Dimensionality Reduction – Chapter 08

This repository contains my complete implementation and study of **Dimensionality Reduction techniques** from the book *Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow*.

The goal of this project is to deeply understand how high-dimensional data can be transformed into lower dimensions while preserving meaningful patterns.

---

# 📌 Topics Covered

This chapter explores the following concepts:

1. The Curse of Dimensionality
2. Main Approaches for Dimensionality Reduction
3. Projection Methods
4. Manifold Learning
5. Principal Component Analysis (PCA)
6. Preserving the Variance
7. Principal Components
8. Projecting Down to d Dimensions
9. PCA Implementation using Scikit-Learn
10. Explained Variance Ratio
11. Choosing the Right Number of Dimensions
12. PCA for Data Compression
13. Randomized PCA
14. Incremental PCA
15. Kernel PCA
16. Kernel Selection and Hyperparameter Tuning
17. Locally Linear Embedding (LLE)
18. Other Dimensionality Reduction Techniques

---

# 🧪 Experiments Performed

The following experiments were implemented:

• PCA from scratch using NumPy  
• PCA using Scikit-Learn  
• Dimensionality reduction for visualization  
• PCA for dataset compression  
• Kernel PCA with different kernels  
• Incremental PCA for large datasets  
• Manifold learning using LLE
